Welcome to the Meowzer API reference documentation.

## SDK Overview

The Meowzer SDK provides a comprehensive set of tools for creating and managing autonomous animated cats in your web projects.

### Main Components

- **Meowzer SDK** - Main entry point and initialization
- **Cat Manager** - Create and manage cat instances
- **Storage Manager** - Persist cats to IndexedDB
- **Interaction Manager** - Handle food, toys, and other interactions
- **Hook Manager** - Lifecycle hooks and events
- **Plugin Manager** - Extend functionality with plugins

## Core Libraries

Meowzer is built on four specialized libraries:

- **MeowKit** - Cat appearance generation and customization
- **Meowtion** - Animation engine and movement
- **MeowBrain** - AI behaviors and decision making
- **MeowBase** - Storage and persistence layer
