## What is Meowzer?

Meowzer is a TypeScript SDK for creating animated cat sprites that wander around web pages with personality-driven AI behaviors. Perfect for adding delightful, interactive companions to websites, games, and creative projects.

## Quick Example

Get a cat on your page in just 4 lines of code:

```typescript
import { Meowzer } from "meowzer";

const meowzer = new Meowzer();
await meowzer.init();

const cat = await meowzer.cats.create({ name: "Whiskers" });
cat.place(document.body);
```

## Key Features

### Customizable Appearance

- Fur colors and patterns (tabby, spotted, tuxedo, calico)
- Eye colors and sizes
- Different body sizes
- Accessories (coming soon)

### AI-Driven Behaviors

- Personality-based decision making
- Environmental awareness
- State machine with multiple behaviors
- Configurable decision frequency

### Rich Interactions

- **Food** - Basic and fancy variants
- **Water** - Hydration system
- **Yarn** - Throwable toys
- **Laser Pointer** - Chase mechanics
- **Custom interactions** - Extend with plugins

### Persistence & Storage

- IndexedDB integration
- Collection management
- Import/export support
- Migration system

## Browser Support

Meowzer works on all modern browsers:

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

Only one runtime dependency: **GSAP** for smooth animations (~70KB total)
