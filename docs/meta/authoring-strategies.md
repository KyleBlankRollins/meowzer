# Authoring strategies and references

- Topic-based authoring
  - Topic types vs document types
    - Topics can be various strictly defined types. Could be DITA, could be Diataxis.
    - Document types are completely arbitrary and should serve reader needs and expectations. They're schemas for assembling topic types in a regular structure.
