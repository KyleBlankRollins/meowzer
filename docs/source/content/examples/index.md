Explore interactive examples of Meowbase and Meowzer in action.

## Available Examples

### [CRUD Operations](/examples/CRUD)

Interactive demo of Meowbase's Create, Read, Update, and Delete operations. Add cats, update their information, and manage collections.

### [Autonomous Cats](/examples/autonomous-cats)

Watch autonomous cats with AI behavior move around on screen using Meowzer. See how different personality types affect behavior.

### [Curved Path Movement](/examples/curved-paths)

See cats move along smooth curved paths using GSAP's MotionPath plugin. Create figure-8 patterns, S-curves, and organic movement.
