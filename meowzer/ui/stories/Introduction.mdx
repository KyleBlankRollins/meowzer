## Quick Start

The fastest way to add cats to your website:

```html
<!-- Add this to any HTML page -->
<script type="module" src="https://cdn.meowzer.dev/ui@1/all.js"></script>
<mb-cat-overlay></mb-cat-overlay>
```

That's it! You now have fully-functional cat animations.

## Component Categories

### Providers
Context providers that manage Meowzer SDK state and make it available to child components.

### Creation Components
UI for creating and customizing cats with appearance, personality, and preview.

### Management Components
Tools for managing active cats - pause, resume, destroy, save to collections.

### Gallery Components
Browse, load, and manage saved cats from collections with import/export functionality.

### Drop-in Solutions
Complete, self-contained components that include everything needed:
- **`mb-cat-overlay`** - Floating overlay with tabs
- **`mb-cat-playground`** - Full sandbox environment
