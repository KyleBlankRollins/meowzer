    Class Meowzer

    Main Meowzer SDK class

    This is the primary entry point for using Meowzer. It provides a unified interface for creating, managing, and persisting animated cats.

    Example

    const meowzer = new Meowzer({
      storage: { adapter: 'indexeddb' }
    });

    await meowzer.init();

    const cat = await meowzer.cats.create({
      name: "Whiskers",
      settings: { color: "orange", pattern: "tabby", ... }
    });

    cat.place(document.body);

    createLaserPointer

    • createLaserPointer(): LaserPointer

      Create a new laser pointer for manual control

      Returns a LaserPointer instance for controlling the laser position. The UI package is responsible for rendering the visual laser dot.

      Phase 1: Manual control only Phase 3: Auto-patterns (circle, zigzag, etc.)

      Returns LaserPointer

      Example

      const laser = meowzer.createLaserPointer();

      // Turn on at position
      laser.turnOn({ x: 500, y: 300 });

      // Move to new position
      laser.moveTo({ x: 600, y: 400 });

      // Listen to movement events
      laser.on("moved", (event) => {
        console.log("Laser at:", event.position);
      });

      // Turn off when done
      laser.turnOff();
